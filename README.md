# cafeOwner

Helping artists find and apply to quality open calls.

## Features

*   Web and Android platforms
*   Browse & Search Open Calls
*   Advanced Filtering (No Vanity Galleries, No Pay-to-Play, Physical Exhibitions Only)
*   User Accounts & Submission Tracking

## Technology Stack

*   **Backend:** Firebase (Firestore, Auth, Storage, Functions)
*   **Web:** React
*   **Android:** Kotlin, Jetpack Compose
*   **Scraper:** Python (Requests, BeautifulSoup)

## Scraping Approach

This project aggregates open calls by scraping various websites. We strive to do this ethically:
*   Respecting `robots.txt` and Terms of Service.
*   Implementing rate limits.
*   Actively filtering out low-quality listings such as:
    *   Vanity galleries
    *   Pay-to-play schemes
    *   Online-only exhibitions
    *   Blacklisted domains like `exhibizone.com`.
